PhotosXX

A JavaFX-based photo management application that allows users to store, organize, and search photos across albums. Built for CS 213 Assignment 3, this project emphasizes GUI design with FXML, object serialization, and GitHub-based collaboration.

Features

Login system with admin and user roles

Admin can create, delete, and list users

Users can:

Create, rename, and delete albums

Add, move, copy, and remove photos

Add and delete tags for each photo

Search photos by date range or tag-value pairs

View photos with captions and dates

Create albums from search results
